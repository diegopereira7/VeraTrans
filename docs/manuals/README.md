# Manuales de usuario — VeraBuy Traductor

Carpeta reservada para manuales dirigidos a **operadores humanos** del
sistema (no a desarrolladores ni a Claude Code).

## Cuándo añadir algo aquí

Cuando se cierre alguna de estas fases del [`roadmap`](../roadmap/roadmap.md):

| Manual | Se genera al cerrar |
|---|---|
| `manual-revision.md` — cómo revisar una factura en la UI, qué significan los badges, cuándo aceptar y cuándo corregir | Paso 7 (UI de revisión con `mark_confirmed`/`mark_corrected`) |
| `manual-operador.md` — flujo diario completo: subir lote → revisar → exportar | cuando se estabilice el flujo de producción |
| `manual-shadow-mode.md` — cómo leer los reportes de modo sombra y qué hacer con las discrepancias | Paso 9 del roadmap |

## Convenciones

- Lenguaje dirigido al operador, no técnico.
- Capturas de pantalla bienvenidas (subcarpeta `manuals/img/`).
- Cada manual empieza con "Cuándo usar este manual" y "Qué no cubre".
- Cualquier manual aquí debe referenciarse desde
  [`docs/README.md`](../README.md) en su tabla principal.
